# pocketMoney
포켓머니는 해외여행 후 남은 돈을 동네 이웃과 환전할 수 있도록 중계해주는 서비스입니다. 사용자가 화폐 종류, 다녀온 나라, 금액 등을 적어 글을 올리면 같은 지역에 살고 있는 사용자를 묶어 나타내줍니다.

메인화면에서는 지역과 나라를 검색할 수 있습니다.
